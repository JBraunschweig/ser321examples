You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /power -- takes two positive integers and returns all powers of number 1 up to number 1 to the power of number 2. Ex: http://35.86.110.65:8080/power?num1=2&num2=12
  • /fib -- takes two positive integers and returns the nth numbers in the fibonanci sequence between the two numbers. Ex: http://35.86.110.65:8080/power?num1=3&num2=6
